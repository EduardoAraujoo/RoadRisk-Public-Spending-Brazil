# Business Understanding (CRISP-DM)

## Stakeholders
- Minas Gerais State Government
- PRF (Federal Highway Police)
- SAMU MG

## Business questions
- How can we link accident type to likely causes (MG-only)?
- Which highways/segments should be prioritized for enforcement to reduce accidents?
- Where should SAMU readiness be increased depending on the time of year?
- How can we reduce public spending while improving safety outcomes?

## Success criteria (draft)
- Clear, actionable hotspots/seasonality insights for MG highways.
- Reproducible ETL pipeline producing MG-only curated datasets.
- If modeling is validated: predictive performance measured with appropriate metrics (to be defined after data review).
